# Colima SwiftBar Monitor

A [SwiftBar](https://github.com/swiftbar/SwiftBar) plugin that shows [Colima](https://colima.run/) VM CPU and memory usage in the macOS menu bar.

## Install

1. Copy `colima-meters.30s.sh` to your SwiftBar plugins directory
2. Make it executable: `chmod +x colima-meters.30s.sh`
